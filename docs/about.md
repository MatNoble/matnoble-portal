# 关于 MatNoble

记录数学教学、交互式课件和相关开发工作。

## 角色定位

我是 **MatNoble**。作为一名**高校数学讲师**，我主要讲授微积分、线性代数等课程；作为一名**独立开发者**，我会用代码制作教学演示和学习工具。

我的教学重点是把概念、计算步骤和几何图像对应起来，尽量减少学生在符号和操作之间的断裂感。

## 技术与教学栈

- **教学领域**: 《高等数学》、《线性代数》、《离散数学》等本科数学课程。
- **技术工具**: 使用 **Python/Matlab** 做数值实验，使用 **Vue/TypeScript** 开发交互式教学工具，使用 **LaTeX** 排版课程材料。
- **研究背景**: 数学系硕士，研究方向包含偏微分方程 (PDE) 数值求解。

## 常见问题 (FAQ)

**Q: MatNoble 的教学特色是什么？**  
**A:** 主要是把计算步骤标准化，并用图形或交互演示辅助说明。例如微分计算、DI 表格法和空间几何演示等内容。

**Q: 为什么这里不仅有数学，还有技术内容？**  
**A:** 因为很多教学材料可以通过程序生成、验证或交互展示。本站会同时记录数学内容和相关工具开发。

**Q: 如何获取本站提到的复习资料？**
