## 1. 图像预处理与阈值分割

在进行图像识别的第一步，我们需要将目标对象（红色的 LED 发光管）从复杂的背景中分离出来，生成二值化掩膜图像。

### 1.1 核心知识点

*   **色彩空间与通道分离**：彩色数字图像通常由 Red (R)、Green (G)、Blue (B) 三个色彩通道叠加而成。针对特定颜色的目标，可通过分离单一通道或通道间差值运算进行凸显。
*   **逻辑矩阵生成与阈值分割**：基于矩阵的比较运算，可以根据像素数值的大小直接生成包含 `logical 0` 和 `logical 1` 的二值化矩阵。

### 1.2 算法实现

由于目标数码管呈现高亮红色，其 R 通道的像素值显著高于 G 通道和 B 通道。我们可以设定合适的色彩差值阈值，将红色区域提取出来。

```matlab
% 读取图像文件
img = imread('digital_clock.png');

% 提取 R、G、B 通道分量
R = img(:,:,1);
G = img(:,:,2);
B = img(:,:,3);

% 色彩差值分割：提取红色分量显著大于绿色和蓝色的区域
% 设定阈值要求红通道绝对值大于 150，且与蓝绿通道的差值大于 50
bw = (R > 150) & (R - G > 50) & (R - B > 50);

figure;
imshow(bw);
title('预处理二值图像');
```

### 闯关题目 1

1. **色彩空间转换**：在复杂的环境光下（如强光反射或严重偏色），直接依赖 RGB 差值运算的鲁棒性较差。请问在 MATLAB 中，应调用什么内置函数将 RGB 图像转换到对光照和颜色更具有分离特性的 HSV 色彩空间？
2. **逻辑运算分析**：假设矩阵 `A = [10 50 100; 200 80 150]`，执行指令 `mask = A > 90` 后，矩阵 `mask` 的数据类型和内容分别是什么？

<details>
<summary>▶ 查看答案</summary>

1. **色彩空间转换**：应使用 `rgb2hsv()` 函数。该函数将图像转换为包含 Hue（色相）、Saturation（饱和度）和 Value（明度）的矩阵，后续可直接通过判定 H 分量是否在红色的色相区间内来实现更稳定的分割。
2. **逻辑运算分析**：`mask` 的数据类型为 `logical`（逻辑型矩阵）。内容为 `[0 0 1; 1 0 1]`。

</details>

## 2. 形态学运算与连通域分析

提取二值化图像后，面临的主要工程挑战是数码管的物理结构问题。七段数码管的相邻线段之间在物理上是断开的。直接对当前的二值图像进行连通域查找，会导致一个完整的数字被拆分为多个零碎的独立区块。

### 2.1 核心知识点

*   **形态学闭运算 (`imclose`)**：形态学中的闭运算是指“先膨胀后腐蚀”的过程。它能够在不明显改变目标区域总面积的前提下，填补目标内部的细小空洞，并“缝合”断裂的微小间隙。
*   **结构元素 (`strel`)**：用于定义形态学运算的邻域范围和形状，如矩形、圆形等。
*   **连通域特征提取 (`regionprops`)**：MATLAB 用于测量图像区域属性的函数，可快速获取目标的面积、质心坐标以及用于裁切的边界框 (`BoundingBox`)。

### 2.2 算法实现

通过构建合适的结构元素并执行闭运算，我们将同一个数字内断开的段缝合为单一的连通域。

```matlab
% 1. 构造结构元素：采用 15x15 的矩形窗口以覆盖线段间的缝隙
se = strel('rectangle', [15, 15]); 

% 2. 执行闭运算：缝合物理断裂
bw_closed = imclose(bw, se);       

% 3. 噪声过滤：移除面积小于 500 像素的孤立噪点
bw_clean = bwareaopen(bw_closed, 500); 

% 4. 特征提取：获取连通域的边界框 (BoundingBox) 和面积 (Area)
stats = regionprops(bw_clean, 'BoundingBox', 'Area');

figure;
imshow(img); hold on;

% 遍历并在原图上绘制定位框
for i = 1:length(stats)
    bbox = stats(i).BoundingBox;
    % 依据先验知识过滤非目标区域（如中间的冒号，其高度通常较小）
    if bbox(4) > 50  
        rectangle('Position', bbox, 'EdgeColor', 'g', 'LineWidth', 2);
    end
end
title('形态学运算与目标定位结果');
```

### 闯关题目 2

1. **结构元素的尺度**：在上述代码中，如果将结构元素修改为 `se = strel('rectangle', [3, 3])`，最可能导致什么负面结果？
2. **特征属性应用**：除了使用高度 (`bbox(4)`) 过滤中间的冒号外，结合 `regionprops` 提供的信息，还可以使用什么特征参数来剔除面积远小于正常数字的噪点？

<details>
<summary>▶ 查看答案</summary>

1. **结构元素的尺度**：`3x3` 的结构元素尺寸过小，可能无法完全覆盖并“跨越”数码管各段之间的物理缝隙。这将导致闭运算失败，同一个数字依然保持为多个独立的碎片连通域。
2. **特征属性应用**：可以直接使用 `stats(i).Area` 属性。由于冒号的像素面积远小于正常数字，通过设定 `if stats(i).Area > 阈值` 即可有效剔除。

</details>

## 3. 区域采样与译码算法

定位到每一个数字所在的精确区域后，需利用区域图像的数据对目标数字进行判定。

### 3.1 核心知识点

*   **矩阵局部切片**：结合 `BoundingBox` 的坐标与尺寸，使用矩阵索引提取局部图像块。
*   **网格采样与逻辑映射**：模拟底层硬件“七段译码器”的逻辑运算。将数字区域划分为特定的网格或选取特定的坐标系点阵。通过判定这 7 个对应区域（A 至 G 段）内的像素白点均值是否高于设定阈值，生成一个形如 `[1 1 1 1 1 1 0]` 的一维逻辑数组，并映射为具体的十进制数字。

### 3.2 算法实现思路

由于要求学生独立完成该核心算法模块，在此提供算法伪代码与设计指引：

```matlab
% 提取单个数字图像块的伪代码
% bbox 的格式为 [x, y, width, height]
digit_img = bw_clean(round(y) : round(y+height), round(x) : round(x+width));

% 译码器算法指引：
% 1. 可将 digit_img 均分为 3 行 3 列的网格。
% 2. 分别计算顶部、左上、右上、中部、左下、右下、底部特定网格内的像素均值。
% 3. 超过阈值判为 1，否则判为 0，生成包含 7 个元素的数组 code。
% 4. 使用 switch-case 结构，若 code == [1,1,1,1,1,1,0]，则判定结果为 0。
```
