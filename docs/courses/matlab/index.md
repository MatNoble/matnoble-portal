# MATLAB 编程与工程实践

本页面为 MATLAB 强化训练与编程实践的参考教程。课程以计算思维与算法实现为主线，帮助学生建立从数学建模到软件编写、从算法逻辑到系统集成的工程思维。

---

## 1. 课程总体设计

课程采用项目驱动的学习方式，内容按照工程能力递进式设计，从基本的命令行指令与矩阵操作，过渡到数据可视化绘图、控制流与自定义函数编写，并最终通过项目实践完成交互计算器与 GUI 界面开发的闭环。

| 章节 | 教学主题 | 核心技能 | 项目实践 |
| :--- | :--- | :--- | :--- |
| **第一部分** | [MATLAB 基础与矩阵操作](./basics) | 变量与数据类型、矩阵表示、算术运算、切片与逻辑索引 | 学生成绩数据分析 |
| **第二部分** | [MATLAB 数据可视化与绘图](./plotting) | 二维绘图、多图叠加、子图划分、散点与柱状图 | 常用工程图表绘制 |
| **第三部分** | [M 文件与自定义函数规范](./conventions) | 脚本与函数编写、代码规范、控制流结构 | 命令行矩阵计算器实现 |
| **第四部分** | [项目实践：GUI 界面矩阵计算器](./project-gui) | App Designer 界面设计、回调函数绑定、界面与逻辑分离 | 界面封装矩阵计算器实现 |

## 2. 系列教程与学习参考

点击以下章节链接进入对应的学习参考资料：

### [1. MATLAB 基础与矩阵操作](./basics)
介绍 MATLAB 工作区与基础指令、变量命名与数据类型、特殊矩阵生成、行列切片取值、普通乘法与逐元素点乘的区分，以及利用逻辑索引高效过滤数据的方法。本章最后通过“学生成绩数据分析”项目巩固矩阵实战技巧。

### [2. MATLAB 数据可视化与绘图](./plotting)
系统介绍 MATLAB 的工程数据可视化能力。涵盖了 `plot` 函数的基础绘图流程与图表修饰技巧、利用 `hold on` 对比叠加多条折线、使用 `subplot` 划分窗口渲染多个独立子图，以及绘制散点图、柱状图等常用扩展图表。

### [3. M 文件与自定义函数规范](./conventions)
介绍脚本与函数的区别、如何规范地编写自定义函数、文件命名规则、代码规范、控制流（if/for/while）以及 try-catch 防御性设计。本章最后通过实现一个交互式的“命令行矩阵计算器”进行综合实训。

### [4. 项目实践：GUI 界面矩阵计算器](./project-gui)
指导如何使用 App Designer 设计一个图形用户界面，将前置课程中的运算和求解逻辑封装进界面的按钮回调中，实现数据输入、运算选择、求解执行以及结果展示的图形化交互。

## 3. 实践课学习建议
*   **重在代码调试**：建议利用 MATLAB 编辑器中的断点和单步执行工具，观察工作区（Workspace）中变量维度和数值的变化，加深对矩阵运算特性的理解。
*   **遵循编码规范**：保持良好的变量命名规范和函数注释习惯，有助于在编写复杂算法时理清思路，并在团队协作中大幅降低沟通成本。
