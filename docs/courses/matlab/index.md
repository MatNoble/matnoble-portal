# MATLAB 编程与工程实践

本页面为 MATLAB 强化训练与编程实践的参考教程。课程以计算思维与算法实现为主线，帮助学生建立从数学建模到软件编写、从算法逻辑到系统集成的工程思维。

---

## 1. 课程总体设计

课程采用项目驱动的学习方式，内容按照工程能力递进式设计，从基本的命令行矩阵操作与绘图，过渡到代码规范、自定义函数编写，并最终通过两个实践项目完成命令行交互计算器与 GUI 界面计算器的开发。

| 章节 | 教学主题 | 核心技能 | 项目实践 |
| :--- | :--- | :--- | :--- |
| **第一部分** | [MATLAB 基础与绘图](./basics) | 矩阵表示、算术与点运算、切片索引、plot 绘图 | 绘制常见初等函数图像 |
| **第二部分** | [M 文件与函数规范](./conventions) | 脚本与函数编写、变量作用域、编码与命名规范 | 编写通用矩阵计算函数 |
| **第三部分** | [项目实践一：命令行矩阵计算器](./project-calculator) | 命令行菜单交互、try-catch 异常处理、线性方程组求解 | 命令行矩阵计算器实现 |
| **第四部分** | [项目实践二：GUI 界面矩阵计算器](./project-gui) | App Designer 界面设计、回调函数绑定、界面与逻辑分离 | 界面封装矩阵计算器实现 |

## 2. 系列教程与学习参考

点击以下章节链接进入对应的学习参考资料：

### [1. MATLAB 基础与矩阵操作](./basics)
介绍 MATLAB 工作区与基础指令、变量命名与数据类型、等差数列与特殊矩阵生成、行列切片取值、普通乘法与逐元素点乘的条件区别、左除（反斜杠）求解方程组，以及利用 `plot` 绘制基础初等函数。

### [2. M 文件与自定义函数规范](./conventions)
介绍脚本与函数的区别、如何规范地编写自定义函数、M 文件与主函数命名的绑定规则、变量驼峰命名与常量规范、函数 H1 描述行与帮助文档编写标准、局部工作区及子函数设计、分号的使用与 try-catch 防御性设计。

### [3. 项目实践一：命令行矩阵计算器](./project-calculator)
指导如何利用控制流（while-switch）和 try-catch 构建一个健壮的命令行交互式计算器，实现矩阵的输入与生成、基本代数运算、转置、行列式、秩、逆矩阵分析，以及对超定/欠定/方阵方程组的 `\` 高效求解。

### [4. 项目实践二：GUI 界面矩阵计算器](./project-gui)
指导如何使用 App Designer 设计一个图形用户界面，将命令行计算器所实现的运算逻辑和求解逻辑封装进界面的按钮回调中，实现数据输入、运算选择、求解执行以及结果展示的图形化交互。

## 3. 实践课学习建议
*   **重在代码调试**：建议利用 MATLAB 编辑器中的断点和单步执行工具，观察工作区（Workspace）中变量维度和数值的变化，加深对矩阵运算特性的理解。
*   **遵循编码规范**：保持良好的变量命名规范和函数注释习惯，有助于在编写复杂算法时理清思路，并在团队协作中大幅降低沟通成本。
