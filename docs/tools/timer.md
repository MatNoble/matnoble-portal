## ✨ 功能特性

### 🕰️ 精准时钟 (Clock)
极简主义设计的数字时钟，让您时刻保持对当下的感知。以优雅的排版清晰展示当前日期与时间。

### 🍅 番茄专注 (Pomodoro)
利用经典的 **番茄工作法 (Pomodoro Technique)** 提升效率：
- **专注模式**：25 分钟的高强度工作。
- **休息模式**：5 分钟的短暂充电。
- 环形进度条可视化展示剩余时间，助您心无旁骛。

### ⏳ 灵活计时 (Timer)
需要一个快速倒计时？无论是会议控制、头脑风暴还是午间小憩，只需输入分钟数，即可快速开始。
