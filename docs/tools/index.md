# 数学辅助工具

我相信技术应当服务于思维。以下是我开发的辅助数学学习与教学的工具集。

## 在线工具

### [Memorize 记忆助手](./memorize)
一个专注于数学概念与公式记忆的轻量级工具。
- **状态**: ✅ 已上线 (v1.0)
- **核心功能**: 艾宾浩斯遗忘曲线复习、LaTeX 公式原生支持。
- **适用场景**: 考研数学公式记忆、常用定义背诵。

## 开源项目

除了在线服务，我也会开源部分教学使用的脚本与 CLI 工具。请关注我的 [GitHub](https://github.com/matnoble) 获取最新动态。
