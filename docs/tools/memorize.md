# Memorize 记忆助手

> "理解不能代替记忆，但记忆可以辅助理解。"

**Memorize** 是一个极简的 Web 应用，旨在解决数学学习中“公式繁多、容易遗忘”的问题。

## 功能特性

- **原生 LaTeX 支持**: 完美渲染复杂的数学公式。
- **间隔重复算法 (SRS)**: 基于优化的 SM-2 算法，自动安排复习时间。
- **纯净体验**: 无广告，无社交干扰，专注于知识本身。

## 使用指南

1.  **创建卡片**: 前面输入问题（如“泰勒展开通项”），背面输入 LaTeX 代码。
2.  **每日复习**: 根据系统提示完成当天的复习任务。
3.  **量化进步**: 查看记忆留存率统计图表。
