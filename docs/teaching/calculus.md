# 工科微积分 (Calculus)

*本文档仅作教学结构说明，不包含具体习题。*

## 课程模块

1.  **一元微积分**
    - 极限与连续性的工程意义
    - 导数：变化率与线性逼近
    - 积分：累积量与微分方程初步
2.  **多元微积分**
    - 偏导数与梯度场
    - 重积分与坐标变换
    - 场论初步（散度与旋度）

## 推荐教材
- 同济大学《高等数学》
- James Stewart, *Calculus*
