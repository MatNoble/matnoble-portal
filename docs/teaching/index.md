# 大学数学教学体系

欢迎来到 MatNoble 的数学教学实验室。在这里，数学不再是枯燥的公式堆砌，而是可以被拆解、被可视化、被逻辑推演的**思维架构**。

## 🗺️ 学习路径图 (Learning Roadmap)

为了帮助你更高效地通关大学数学，我建议按照以下逻辑进行探索：

### 1. [微积分：从直觉到实战](./calculus)
- **新手村**：极限思想的本质（不再纠结于 $\varepsilon-\delta$）。
- **进阶攻略**：**[微分万能公式](./derivative-method)** —— 彻底搞定复合函数、隐函数求导。
- **高阶战术**：**[核心心法速查表](./cheatsheet)** —— 收录 DI 表格法、积分常用替换、泰勒展开等必背技巧。

### 2. [线性代数：几何直觉重构](./linear-algebra)
- **底层逻辑**：忘记繁琐的行列式计算，从**线性变换**与**基变换**理解矩阵。
- **核心版图**：矩阵分解（SVD, Eigenvalues）在数据科学中的实际映射。

## 💎 核心方法论 (The MatNoble Way)

我不提倡“题海战术”，而是主张**“武器库升级”**。

- **微分万能公式 ($dy = f'(\square)d\square$)**：利用微分形式不变性，将链式法则简化为剥洋葱式的填空操作。适用于隐函数、参数方程求导，是**考研数学**中的提速利器。
- **DI Method (表格积分法)**：针对分部积分法的系统性优化。通过 $D$ (求导) 和 $I$ (积分) 两列快速得出结果，让复杂的积分过程变得清晰且不易出错。
- **可视化教学**：使用动画与交互式工具，将抽象的 $\mathbb{R}^n$ 空间具象化。
