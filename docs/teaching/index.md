# 大学数学教学体系

作为高校数学教师，我构建了一套以“**直觉优先，严谨支撑**”为核心的教学体系。本体系不仅关注考试成绩，更致力于通过数学训练逻辑思维，将抽象的数学工具转化为解决实际问题的利器。

## 核心课程

### [微积分 (Calculus)](./calculus)
面向本科生的微积分课程，涵盖极限、导数、积分及多元微积分。
- **教学重点**：极限思想、微分应用、多元微积分的可视化理解。
- **特色**：引入 Python/Matlab 进行数值实验，连接物理与工程背景。

### [线性代数 (Linear Algebra)](./linear-algebra)
从几何直观出发的现代线性代数，带你重新认识矩阵。
- **教学重点**：线性空间、线性变换、矩阵分解（SVD）。
- **特色**：摒弃传统的“行列式优先”讲法，强调矩阵作为变换的本质。

## 教学特色与方法论

1.  **直观化 (Visualization)**: 坚持“能画图的绝不只推公式”，通过几何直觉降低认知门槛。
2.  **工具化 (Tooling)**: 鼓励使用现代计算工具（Mathematica, Python, LaTeX）来验证猜想和辅助学习。
3.  **结构化 (Structure)**: 建立知识图谱，通过 [微分万能公式](./derivative-method) 等提炼，将知识点串联成网。
4.  **实战化 (Practical)**: 提供 [核心心法速查表](./cheatsheet)，针对期末考试与考研数学进行精准复习指导。
