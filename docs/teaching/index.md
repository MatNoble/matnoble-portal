## 主要方法

这些页面主要围绕以下方法展开：

- **微分万能公式 ($dy = f'(\square)d\square$)**：利用微分形式不变性，把链式法则写成逐层微分的过程。适用于复合函数、隐函数和参数方程求导。
- **DI Method (表格积分法)**：分部积分法的一种表格化写法。通过 $D$ (求导) 和 $I$ (积分) 两列组织计算步骤。
- **可视化教学**：使用动画与交互式工具辅助理解几何对象和矩阵变换。
