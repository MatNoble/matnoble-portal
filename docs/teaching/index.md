# 大学数学教学体系

作为面向工科专业的数学教师，我构建了一套以“**直觉优先，严谨支撑**”为核心的教学体系。

## 核心课程

### [微积分 (Calculus)](./calculus)
面向工科本科生的微积分课程。
- **教学重点**：极限思想、微分应用、多元微积分的可视化理解。
- **特色**：引入 Python/Matlab 进行数值实验，连接物理与工程背景。

### [线性代数 (Linear Algebra)](./linear-algebra)
从几何直观出发的现代线性代数。
- **教学重点**：线性空间、线性变换、矩阵分解（SVD）。
- **特色**：摒弃传统的“行列式优先”讲法，强调矩阵作为变换的本质。

## 教学理念

1.  **直观化 (Visualization)**: 能画图的绝不只推公式。
2.  **工具化 (Tooling)**: 掌握现代计算工具（Mathematica, Python）验证猜想。
3.  **结构化 (Structure)**: 建立知识图谱，而非孤立的考点记忆。
